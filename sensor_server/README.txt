This is Hakoirimusume Sensor Server application.

This application will provide camera & sensor data for Hakoirimusume application.

This application depends on only pre-installed modules on Raspberry Pi OS.
Thus, you may not need to create venv or install any packages via pip.

All you need to do is editing `config.yml` and running `python3 main.py`.

If you need further information, please refer to `../docs/sensor_server.md`.
